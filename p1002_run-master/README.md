# 숨좀쉬자(PleaseBreath) - Server
> 공기 청정 기기를 제어하여 실내 환경을 쾌적하게 만드는 AI　공기 관리 솔루션

![pb](./image/pb.png)

## 실행 방법
```sh
cd src
npm start
```

## 개발 내용
> - 라즈베리파이에서 전송한 실내, 실외 공기 데이터를 웹 앱에 적용 
  라즈베리파이와 Http 통신
